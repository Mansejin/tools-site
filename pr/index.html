Setup

Adobe Premiere Pro 영문 설치법

강의 실습을 위해 English (International) 버전으로 설치해주세요.

1

Adobe 계정 만들기 & Creative Cloud 설치

adobe.com/kr에 접속해 회원가입 후 로그인합니다. Creative Cloud 앱을 설치하세요. Creative Cloud는 프리미어, 애프터이펙트, 포토샵, 일러스트레이터 등 Adobe 프로그램을 관리하는 통합 앱입니다.

2

기본 설치 언어를 English로 변경 중요

Creative Cloud 실행 후, 왼쪽 위 ≡ (삼선) 메뉴 → 파일 → 환경 설정 (단축키 Ctrl + ,)으로 진입합니다.

  1. 좌측 언어 메뉴 클릭
  2. 기본 설치 언어 → 언어를 한국어에서 English (International)로 변경
  3. 완료 클릭
왜 영문인가요? 강의 자료와 튜토리얼이 영문 UI 기준으로 진행됩니다. 한국어로 설치하면 메뉴 위치가 달라져 따라하기 어려울 수 있습니다.
3

Premiere Pro 설치

Creative Cloud 앱 메뉴에서 Premiere Pro를 찾아 설치합니다. 설치가 완료되면 수업 전에 한 번 실행해 정상 작동을 확인해주세요.
