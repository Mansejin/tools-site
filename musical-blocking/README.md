# StageCue — 뮤지컬 동선 체크리스트

가사·큐·대사 단위로 배역 동선을 기록하고, 박자(beat) 기반 키프레임으로 편집·재생하는 웹 앱입니다.

## 기능

- **대본 업로드 / 붙여넣기**: `.txt` 업로드 또는 Ctrl+Enter로 적용
- **대사·글자 선택**: 줄을 클릭하거나 글자를 드래그 선택 → 큐로 사용
- **무대 동선**: 배역 마커를 드래그하면 현재 큐·박에 키프레임 저장
- **박자 키프레임**: 동선은 박(beat) 단위로 저장, 타임라인에서 재생·편집
- **넘버 · 템포 맵**: 넘버마다 / 넘버 안 BPM 변화. 재생만 템포 맵을 따르고 키프레임은 유지
- **배역 표시 ON/OFF**: 무대에서 배역별로 토글
- **설정 탭**: 작품·배역·무대·넘버·템포 맵
- **로컬 저장**: `localStorage`에 자동 저장

## 대본 형식 예시

```
【넘버: 바람이 불어오면 / BPM 120】
민수: 여기가… 네가 말한 그 자리야?
지아: 응. 기억나지?

♪ 지아
바람이 불어오면

【BPM 144】
※ 템포 업 — 후렴
```

- `이름: 대사` → 대사/가사
- `【넘버: 제목 / BPM 120】` → 넘버 시작 + 템포
- `【BPM 144】` → 넘버 안 템포 변경
- `【…】` / `[…]` → 큐
- `※` / `▶` → 무대지시
- `♪` 시작 줄 → 가사
