# Mindtree

꿈, 미래, 업무, 가족, 대인관계, 깊은 생각, 철학 — 흩어진 생각을 트리로 정리하는 웹앱.

**서비스 URL:** https://mansejin.com/mindtree/

## 개발

```bash
cd mindtree-app
npm install
npm run dev
# http://localhost:5180/mindtree/
```

## 빌드 & 배포

```bash
npm run build
# → ../mindtree/ (GitHub Pages 정적 배포)
```

## 구조

- `mindtree-app/` — 소스 코드 (Vite + React)
- `mindtree/` — 빌드 결과물 (`mansejin.com/mindtree/`)
